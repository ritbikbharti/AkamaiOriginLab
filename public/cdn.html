Each endpoint returns a different Cache-Control directive so you can validate cache hits, TTL expiry, and bypass behavior at the edge.

Every call below also sends the Akamai debug Pragma request headers, so when the request transits an Akamai property the response surfaces X-Cache, X-Check-Cacheable, and X-Cache-Key. Hitting the origin directly returns an empty value for these.

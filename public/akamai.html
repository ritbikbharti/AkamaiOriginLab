Reference: Akamai request headers

True-Client-IP
Original client IP, when "True Client IP Header" behavior is enabled.
X-Forwarded-For
Standard chain of forwarded IPs; Akamai appends the edge IP.
X-Akamai-Edgescape
Comma-separated geo / network attributes (e.g. country_code=US,region_code=CA,city=SANJOSE,...).
Pragma
Debug directives such as akamai-x-cache-on, akamai-x-get-cache-key.
Akamai-Origin-Hop
Number of Akamai hops the request has traversed.
Via
Proxy chain identifier; contains AkamaiGHost for traffic via Akamai.
